Features

  1. Normal API for beginners.
  2. API with Auth Token with complete login, register, and other features for intermediate users.

Login

You can login via either email, username or phonrNumber with password combination.
Endpoint: /login

Movie

Retrieve a list of all movies.
Endpoint: /movies
Retrieve a specific movie by ID.
Endpoint: /movies/:id
Create a new movie.
Endpoint: /movies
Update an existing movie by ID.
Endpoint: /movies/:id
Delete a movie by ID.
Endpoint: /movies/:id

User

Retrieve a list of all users.
Endpoint: /users
Retrieve a specific user by ID.
Endpoint: /users/:id
Create a new user.
Endpoint: /users
Update an existing user by ID.
Endpoint: /users/:id
Delete a user by ID.
Endpoint: /users/:id
